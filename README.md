# Library Book Management (C++)

A console-based Library Book Management System built using C++. This project allows users to add and view books using a simple menu-driven interface.

## 🔧 Features
- Add Book
- View Books
- Menu-driven program
- Dynamic storage using vector

## 🧠 Concepts Used
- C++ Basics
- Vectors (Dynamic Arrays)
- Loops and Conditions
- User Input Handling

## 🚀 How to Run

```bash
g++ library.cpp -o library
./library
```
